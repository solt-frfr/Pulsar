# Pulsar
 A successor to Super Smash Bros. Ultimate's mod manager Quasar, written in C# and Visual Studio.

## Current Features
 Toggleable Tabs.
    Tabs:
        Mods
        ~~Assign~~
        ~~Settings~~
        ~~Download~~
 One instance only.
 Shows an alert when opened through a URL.
 Built-in Logging and Console.
 Dummy Package Builder.
